FUNCTIONS IN JAVASCRIPT
============================================================================================================================

What is a Function?
• Reusable block of code designed to perform a specific task
• Executed only when called (invoked)
• Reduces repetition
• Improves readability and organization
• Mini-program inside main program


--------------------------------------------------------------------------------------------------------------
1) FUNCTION DEFINITION vs FUNCTION CALL
--------------------------------------------------------------------------------------------------------------

Concept            | Syntax                                      | Example
--------------------------------------------------------------------------------------------------------------
Definition         | function functionName(){ }                  | function hello(){     console.log("hello"); }
Calling            | functionName();                             | hello();  // hello

Complete Example:
function greet(){ console.log("Hello, World!"); }
greet();   // Hello, World!

Key Point:
Definition → creates function
Calling → executes function


--------------------------------------------------------------------------------------------------------------
2) FUNCTIONS WITH ARGUMENTS
--------------------------------------------------------------------------------------------------------------

Definition: Arguments are values passed to function.

Syntax                                    | Example
--------------------------------------------------------------------------------------------------------------
function funcName(arg1,arg2){ }           | function printInfo(name,age){
                                          |   console.log(`${name}'s age is ${age}.`);
                                          | }

Call Example:
printInfo("Ajay",25);


--------------------------------------------------------------------------------------------------------------
3) RETURN KEYWORD
--------------------------------------------------------------------------------------------------------------

Definition:
return sends value back as output of function.

Concept Flow:
input  →  [ Function ]  →  output

Syntax                                    | Example
--------------------------------------------------------------------------------------------------------------
function func(a,b){ return value; }       | function sum(a,b){ return a+b; }

let result = sum(5,3);   // 8

SCOPE IN JAVASCRIPT
============================================================================================================================

Definition:
Scope determines accessibility (visibility) of variables, functions, objects.

Types:
1) Global Scope
2) Function Scope
3) Block Scope
4) Lexical Scope


--------------------------------------------------------------------------------------------------------------
1) GLOBAL SCOPE
--------------------------------------------------------------------------------------------------------------

• Declared outside any function/block
• Accessible everywhere

Example:
let name="Ajay";

function greet(){ console.log(name); }
greet();
console.log(name);


--------------------------------------------------------------------------------------------------------------
2) FUNCTION SCOPE
--------------------------------------------------------------------------------------------------------------

• Declared inside function
• Accessible only inside that function

Example:
function calSum(a,b){
    let sum=a+b;
    console.log(sum);
}

calSum(5,3);
console.log(sum);   // Error


--------------------------------------------------------------------------------------------------------------
3) BLOCK SCOPE
--------------------------------------------------------------------------------------------------------------

• let / const inside { }
• Accessible only inside block
• Applies to if, for, while etc.
• var does NOT follow block scope

Example:
if(true){
    let x=10;
    console.log(x);
}
console.log(x);   // Error


--------------------------------------------------------------------------------------------------------------
4) LEXICAL SCOPE
--------------------------------------------------------------------------------------------------------------

• Inner functions can access outer variables
• Scope determined by where function is written
• Lookup flows outward → Parent → Global

Example:
function outer(){
    let message="Hello";

    function inner(){
        console.log(message);
    }

    inner();
}

outer();

Important Rule:
Inner function CAN access outer variables
Outer function CANNOT access inner variables


--------------------------------------------------------------------------------------------------------------
SCOPE SUMMARY
--------------------------------------------------------------------------------------------------------------

Scope Type   | Accessibility
--------------------------------------------------------------------------------------------------------------
Global       | Accessible everywhere
Function     | Only inside function
Block        | Only inside { }
Lexical      | Inner can access outer variables

🔹 Core Function Types (Must Know)

1) Function Declaration
function greet() {}

2) Function Expression
const greet = function() {};

3) Arrow Function
const greet = () => {}

---------------------------------------------------------------

🔹 Important Variations (Also Used)

4) Anonymous Function (no name)
setTimeout(function() {}, 1000);

5) Callback Function (passed as argument)
arr.forEach(x => console.log(x));

6) IIFE (Immediately Invoked Function Expression)
(function() {
  console.log("run");
})();
