============================================================================================================================
JAVASCRIPT IDENTIFIERS
============================================================================================================================

Definition:
Identifiers are names used for variables, functions, classes, or objects in JavaScript.

----------------------------------------------------------------------------------------------------------------------------
RULES
----------------------------------------------------------------------------------------------------------------------------
1) Allowed characters → letters (a-z, A-Z), digits (0-9), underscore (_), dollar ($)
2) Must start with letter, _ or $
3) Cannot start with a digit
4) Case-sensitive (myVar and MyVar are different)
5) Cannot use reserved keywords
6) No spaces or special characters except _ and $
7) Should be meaningful (best practice)

----------------------------------------------------------------------------------------------------------------------------
EXAMPLES
----------------------------------------------------------------------------------------------------------------------------
Valid

var myVar;
let _count;
const $price = 100;
function getData(){}

Invalid

var 1stValue;      // starts with digit
let my-var;        // contains hyphen
const var = 10;    // reserved keyword

----------------------------------------------------------------------------------------------------------------------------
COMMON NAMING STYLES
----------------------------------------------------------------------------------------------------------------------------
camelCase   → myVariable, totalAmount, getUserData
PascalCase  → UserProfile, EmployeeDetails
snake_case  → my_variable, total_amount


============================================================================================================================
BOOLEAN IN JAVASCRIPT
============================================================================================================================

Definition:
Boolean is a logical data type with only two values → true or false.

Used in conditions, loops, and comparisons.

Examples

let isActive = true;
let hasAccess = false;

Returned by comparisons

5 > 3        // true
10 === "10"  // false

Key Points
1) Only two values → true / false
2) Case-sensitive
3) Used in logical expressions and conditions


============================================================================================================================
WHAT IS TYPESCRIPT
============================================================================================================================

Definition:
TypeScript is a superset of JavaScript developed by Microsoft that adds static typing.
It compiles to plain JavaScript.

----------------------------------------------------------------------------------------------------------------------------
KEY FEATURES
----------------------------------------------------------------------------------------------------------------------------
• Static type checking
• Better IDE support (autocomplete, refactoring)
• Supports modern ECMAScript features
• OOP support (classes, interfaces, modules)
• Gradual adoption in existing JS projects

Example

JavaScript
let age = 23;
age = "twenty three";     // allowed

TypeScript
let age: number = 23;
age = "twenty three";     // error


============================================================================================================================
STRINGS IN JAVASCRIPT
============================================================================================================================

Definition:
String is a sequence of characters written inside double/single quotes "", '' or backtics ``.

Strings are primitive and immutable.

+ operator is used for concatenation.

----------------------------------------------------------------------------------------------------------------------------
DECLARATION
----------------------------------------------------------------------------------------------------------------------------

let name = "Ajay";
let city = 'Delhi';
let greeting = `Hello ${name}`;

----------------------------------------------------------------------------------------------------------------------------
KEY FEATURES
----------------------------------------------------------------------------------------------------------------------------

1) Immutable (cannot modify characters)
2) Zero-based indexing
3) Supports many built-in methods

-Indexing means accessing elements or characters using their position number,
works only for reading string and not changing it.

Index access
let str = "JavaScript";
str[0]    // "J"
str[4]    // "S"

Length

str.length


============================================================================================================================
COMMON STRING METHODS
============================================================================================================================

let str = "  Hello World  ";

str.toUpperCase()      // "  HELLO WORLD  "
str.toLowerCase()      // "  hello world  "

str.trim()             // "Hello World"

str.slice(2,7)         // "Hello"
str.substring(2,7)     // "Hello"

str.replace("Wo rld","JS")   // "  Hello JS  "

str.includes("Hello")  // true

str.split(" ")         // ["", "", "Hello", "World", "", ""]

"Hello".concat(" ","JS")  // "Hello JS"

str.charAt(2)          // "H"
str.indexOf("World")   // 8
str.lastIndexOf("l")   // 11

============================================================================================================================
slice() vs substring()
============================================================================================================================

Both extract part of a string.

let str = "HelloWorld";

slice()

str.slice(-5)        // "World"

substring()

str.substring(-5)    // "HelloWorld"   (negative treated as 0)


============================================================================================================================
TEMPLATE LITERALS
============================================================================================================================

Definition:
Template literals use backticks (`) and allow variable interpolation.

Interpolation means inserting variables or expressions inside strings.

Example

let name = "Ajay";

`Hello ${name}`

Uses
• Dynamic strings
• String formatting
• Logging and messages


============================================================================================================================
STRING IMMUTABILITY
============================================================================================================================

Definition:
Strings are immutable → characters cannot be changed after creation.

Example

let str = "hello";

str[0] = "H";

console.log(str);

Output
hello

Reason:
JavaScript does not modify the original string.  
Instead it creates a new string.

Example

let str = "hello";

str = str + " world";

console.log(str);

Output
hello world


============================================================================================================================
STRING METHODS AND IMMUTABILITY
============================================================================================================================

Methods return a NEW string and do not modify the original.

Example

let str = "hello";

let result = str.toUpperCase();

console.log(str);      // hello
console.log(result);   // HELLO


============================================================================================================================
STRING vs ARRAY
============================================================================================================================

Strings → Immutable  
Arrays  → Mutable

Example

let arr = [1,2,3];
arr[0] = 10;

console.log(arr);   // [10,2,3]


let str = "abc";
str[0] = "z";

console.log(str);   // "abc"

============================================================================================================================
NULL AND UNDEFINED
============================================================================================================================

undefined
• Variable declared but not assigned value

let x;
console.log(x);   // undefined

null
• Intentional absence of value

let y = null;

Key Points

typeof undefined   // "undefined"
typeof null        // "object"

undefined → not assigned  
null → intentionally empty value

Note:
typeof null returning "object" is a historical JavaScript bug.

============================================================================================================================
TRUTHY AND FALSY VALUES
============================================================================================================================

Every value in JavaScript behaves as either truthy or falsy in boolean context.

Example context
if, &&, ||, !

----------------------------------------------------------------------------------------------------------------------------
FALSY VALUES (ONLY 7)
----------------------------------------------------------------------------------------------------------------------------

false
0
-0
0n
""
null
undefined
NaN

Example

if (0) {
    console.log("Runs");
} else {
    console.log("Falsy");
}

----------------------------------------------------------------------------------------------------------------------------
TRUTHY VALUES
----------------------------------------------------------------------------------------------------------------------------

All other values are truthy.

true
1
-5
"Hello"
"0"
[]
{}
function(){}

Example

if ("Hello") {
    console.log("Truthy");
}

----------------------------------------------------------------------------------------------------------------------------
CONFUSING CASES
----------------------------------------------------------------------------------------------------------------------------

Boolean("0")   // true
Boolean([])    // true
Boolean({})    // true

Rule:
Only the 7 listed values are falsy.  
Everything else is truthy.
