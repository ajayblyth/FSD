
ALERT & PROMPT (JS)
------------------------------------------------
1. alert(message)
- Shows a pop-up with message.
- Only for displaying info.
- No input from user.
alert("Login successful");

2. prompt(message, default)
- Shows a pop-up with message and input box.
- Returns user input as string.
- Can have optional default value.
- If cancelled → returns null.
👉 Example: let name = prompt("Enter your name", "Guest");

STRING METHODS (JS) - SYNTAX & EXAMPLES
------------------------------------------------
let str = "  Hello World  ";

1. length
// Syntax: str.length
console.log(str.length);        // 15

2. toUpperCase(), toLowerCase
// Syntax: str.toUpperCase(), str.toLowerCase()
console.log(str.toUpperCase()); // "  HELLO WORLD  "
console.log(str.toLowerCase()); // "  hello world  "

3. charAt(index)
// Syntax: str.charAt(index)
console.log(str.charAt(2));    // "H"

4. indexOf(substring), lastIndexOf(substring)
// Syntax: str.indexOf(sub), str.lastIndexOf(sub)
console.log(str.indexOf("o"));  // 5
console.log(str.lastIndexOf("o")); // 7

5. slice(start, end)    //Returns the part of the original string as a new string.
// Syntax: str.slice(start, end)
console.log(str.slice(4));  // starting from 4th index to right

console.log(str.slice(2, 7));  // "Hello", starts from index, stops one before than end.

--str.slice(-num) = str.slice(length-num)
prints values from last, -1 will give last one value, -2 will give last 2 values and so on.

6. substring(start, end)
// Syntax: str.substring(start, end)
console.log(str.substring(2, 7)); // "Hello"

7. substr(start, length)
// Syntax: str.substr(start, length)
console.log(str.substr(2, 5));  // "Hello"

Note:// 
KEY DIFFERENCE sustring() vs substr()
// =========================================

// substring(start, end)
// → second parameter is END INDEX

// substr(start, length)
// → second parameter is LENGTH

8. includes(substring)
// Syntax: str.includes(substring)
console.log(str.includes("World")); // true

9. startsWith(substring), endsWith(substring)
// Syntax: str.startsWith(sub), str.endsWith(sub)
console.log(str.startsWith("  He")); // true
console.log(str.endsWith("ld  "));   // true

// IMPORTANT NOTES
// =========================================

// ✔ Case-sensitive
// ✔ Does NOT search whole string
// ✔ Only checks beginning or ending
// ✔ Returns true or false (boolean)

// Optional Position Parameter

let text = "JavaScript";

console.log(text.startsWith("Script", 4));
// true (starts checking from index 4)

console.log(text.endsWith("Java", 4));
// true (checks first 4 characters: "Java")

10. trim()
// Syntax: str.trim()
console.log(str.trim());          // "Hello World"


11. split(separator)
// Syntax: str.split(separator)
console.log(str.trim().split(" ")); // ["Hello","World"]

12. replace(searchValue, newValue)
search a value in the string & returns a new string with the value replaced.
// Syntax: str.replace(searchValue, newValue)
console.log(str.replace("World","JS")); // "  Hello JS  "

Note: only first occurence will be replaced.

13. replaceAll(searchValue, newValue)
- Replaces **all occurrences** of searchValue.
- Can use string or global regex.
- Returns new string (original unchanged).

Example:
let str = "Hello World World";
console.log(str.replaceAll("World","JS")); 
// Output: "Hello JS JS"

14. repeat(count)
// Syntax: str.repeat(count)
console.log("Hi ".repeat(3));    // "Hi Hi Hi "

ARRAYS (JS)
------------------------------------------------
Definition:
- An array is an ordered collection of values, where items maintain insertion order and are accessed using 0 based index.
- Stores multiple values in a single variable.
- Indexed (starts from 0).
- Dynamic size (auto-resizable).
- Mutable.
- Allows duplicate values.
- Can store mixed data types.
- Supports random access via index.

Syntax:
let arr = [value1, value2, value3];

- inefficient way to add values one by one:
let num1=10;
let num2=20;
let num3=30;

- efficient way using arrays
Example:
let nums = [10, 20, 30];
console.log(nums[0]);   // 10
console.log(nums.length); // 3

Common Methods:
1. push(value)      → Adds at end
   nums.push(40);   // [10,20,30,40]

2. pop()            → Removes from end
   nums.pop();      // [10,20,30]

3. unshift(value)   → Adds at beginning
   nums.unshift(5); // [5,10,20,30]

4. shift()          → Removes from beginning
   nums.shift();    // [10,20,30]

5. indexOf(value)   → Returns index
   nums.indexOf(20); // 1

6. includes(value)  → true/false check
   nums.includes(30); // true

7. slice(start,end) → Extracts part
   nums.slice(0,2); // [10,20]

8. splice(start,count) → Add/Remove elements
→splice(start, deleteCount, item1, item2, ...)

→ Used to add, remove, or replace elements in an array
→ Modifies original array (in-place)
→ Returns removed elements

Syntax:
array.splice(startIndex, deleteCount, newItems...)

Example 1 – Remove element

let nums = [10, 20, 30, 40];
nums.splice(1, 1);

startIndex = 1 → start from index 1 (value 20)
deleteCount = 1 → remove 1 element

Result:
nums = [10, 30, 40]
Returned value = [20]

👉 Removes from the given start index
👉 Removes consecutive elements (not searching by value)

Example 2 – Remove multiple elements

nums.splice(1, 2);

Removes 2 elements starting from index 1

If nums = [10, 20, 30, 40]
Result → [10, 40]

Example 3 – Add elements

nums.splice(1, 0, 25);

deleteCount = 0 → nothing removed
25 is inserted at index 1

Result → [10, 25, 20, 30, 40]

Example 4 – Replace elements

nums.splice(1, 1, 99);

Removes 1 element at index 1
Inserts 99 at same position

Result → [10, 99, 30, 40]

Important Points

• Works based on index (not value)
• Negative index allowed
nums.splice(-1, 1) → removes last element
• Changes original array
• Returns array of removed elements

Notes:
- Arrays can store mixed data types.
- Arrays are mutable (can be modified).
ARRAY (JS) – INTERVIEW NOTES
------------------------------------------------
Definition:
- Ordered, indexed collection of elements.
- Index starts from 0.
- Dynamic size (auto-resizable).
- Mutable.

Key Features:
- Allows duplicate values.
- Can store mixed data types.
- Continuous memory (conceptually).
- Supports random access via index.

Time Complexity:
- Access by index → O(1)
- Update by index → O(1)
- Search (linear) → O(n)
- Insert at end (push) → O(1)
- Remove from end (pop) → O(1)
- Insert at beginning → O(n)
- Delete from beginning → O(n)
- Insert/Delete in middle → O(n)

Common Use Cases:
- Storing ordered data
- Iteration & traversal
- Stack (push/pop)
- Queue (shift/unshift)

Important:
- Internally optimized but operations requiring shifting cost O(n).
