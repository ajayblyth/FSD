HTML ATTRIBUTES
Attributes give extra information about an element.
Examples: <u> (underline), <i> (italic)


INLINE vs BLOCK ELEMENTS

Block elements → start on new line, take full width, can contain inline elements
Examples: <div>, <p>, <h1>, <ul>, <section>, <form>

Inline elements → stay in same line, take required width, used inside blocks
Examples: <span>, <a>, <img>, <strong>, <b>

====================== SECTION vs DIV vs SPAN ======================

+---------+-----------+-------------------+-------------------------------------------+
| Element | Meaning   | Display           | Usage                                     |
+---------+-----------+-------------------+-------------------------------------------+
| section | meaningful| block-level       | group related content with clear meaning, |
|         |           |                   | usually with a heading; improves          |
|         |           |                   | structure, SEO, accessibility             |
+---------+-----------+-------------------+-------------------------------------------+
| div     | no meaning| block-level       | grouping and layout purposes only         |
+---------+-----------+-------------------+-------------------------------------------+
| span    | no meaning| inline            | styling or grouping small parts of text   |
|         |           |                   | without breaking the line                |
+---------+-----------+-------------------+-------------------------------------------+


====================== <sub> vs <sup> ==============================

+-------+-------------------------------+----------------------------------+
| Tag   | Usage                         | Example                          |
+-------+-------------------------------+----------------------------------+
| <sub> | subscript (below text), used  | H<sub>2</sub>O → H₂O              |
|       | in chemistry                  | (two hydrogen atoms)             |
+-------+-------------------------------+----------------------------------+
| <sup> | superscript (above text),     | a<sup>2</sup> + b<sup>2</sup>     |
|       | used in math                  | → a² + b²                        |
+-------+-------------------------------+----------------------------------+
