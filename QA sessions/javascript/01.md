1) INCREMENT / DECREMENT OPERATORS
================================================================================================================

Concepts:
Pre Increment  → ++x (increment first, then use value)
Post Increment → x++ (use value first, then increment)

Questions:

let x = 5;
console.log(++x);


console.log(x++);


let x = 5;
console.log(++x);
console.log(x++);
console.log(x);


let a = 5;
let b = a++ + ++a;

console.log(a)
console.log(b)

2) EQUALITY & COMPARISON
================================================================================================================

Concepts:
==   → loose equality (type coercion happens)
===  → strict equality (no type conversion)

Questions:

5 == "5"


"A" < "a"


"2" > "10"


NaN !== NaN



================================================================================================================
3) SHORT CIRCUIT EVALUATION
================================================================================================================

Concepts:

||  → returns first TRUTHY value
&&  → returns first FALSY value


Questions:

true || console.log("hello")


let a = 10;
let b = 0 || 20;
console.log(b)


let a = 10;
let b = 0 && 20;
console.log(b)

5) OPERATOR PRECEDENCE
================================================================================================================

Important Order:

1. **     (Exponent)
2. * / %  (Multiplication / Division / Modulus)
3. + -    (Addition / Subtraction)
4. > < == ===
5. &&
6. ||


Questions:

console.log(2 + 3 * 4);


console.log((2 + 3) * 4);


console.log(10 - 2 + 3);


console.log(10 - (2 + 3));


console.log(5 + 2 * 3 ** 2);
