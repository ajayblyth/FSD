====================================================
DOM NAVIGATION PROPERTIES (OVERVIEW)
====================================================

1) parentElement
----------------------------------------------------
Returns the direct parent element of a node.

Example:
let h4 = document.querySelector('h4');
h4.parentElement;
→ <div class="box">...</div>


2) children
----------------------------------------------------
Returns HTMLCollection of direct child ELEMENTS only
(ignores text nodes, comments).

Example:
let box = document.querySelector('.box');
box.children;
→ HTMLCollection(2) [h4, ul]

box.childElementCount;
→ 2

let h4 = document.querySelector('h4');
h4.children;
→ HTMLCollection []   (no child elements)


3) childElementCount
----------------------------------------------------
Returns number of child elements.

Example:
let ul = document.querySelector('ul');
ul.childElementCount;
→ 3


4) Access Specific Child
----------------------------------------------------
Use index (0-based):

ul.children[2];
→ <li>...</li>


5) previousElementSibling
----------------------------------------------------
Returns previous sibling element (ignores text nodes).

Example:
ul.children[2].previousElementSibling;
→ <li>...</li>

ul.children[1].previousElementSibling;
→ <li>...</li>


6) nextElementSibling
----------------------------------------------------
Returns next sibling element.

Example:
ul.children[1].nextElementSibling;
→ <li>...</li>


7) Real Example with Image
----------------------------------------------------
let img = document.querySelector('img');

img.previousElementSibling;
→ <h1>Spider Man</h1>


8) Modifying Previous Sibling Style
----------------------------------------------------
img.previousElementSibling.style;
→ CSSStyleDeclaration {...}

img.previousElementSibling.style.color = "green";
→ 'green'


====================================================
IMPORTANT NOTES
====================================================
✔ parentElement → go UP
✔ children → go DOWN
✔ previousElementSibling → go LEFT
✔ nextElementSibling → go RIGHT
✔ children returns HTMLCollection (array-like)
✔ Index starts from 0

====================================================
ADDING ELEMENTS IN DOM
====================================================

1) document.createElement('tagName')
----------------------------------------------------
Creates a new element (not added to page yet).

Example:
document.createElement('p');
→ <p></p>

let newP = document.createElement('p');
console.dir(newP);
→ p

newP.innerText = "Hi, I am a new p";
→ 'Hi, I am a new p'


====================================================
ADDING ELEMENT TO PAGE
====================================================

2) appendChild(element)
----------------------------------------------------
Adds element as LAST child of parent.
Returns the added element.
Moves element if it already exists somewhere else.

Example:

let body = document.querySelector('body');
body.appendChild(newP);
→ <p>Hi, I am a new p</p>


IMPORTANT:
If you append same element again,
it MOVES (not copies).

Example:

let box = document.querySelector('.box');
box.appendChild(newP);
→ <p>Hi, I am a new p</p>

(Paragraph moved from body to .box)


====================================================
3) append(element)
----------------------------------------------------
Adds element as LAST child.
More flexible than appendChild.

✔ Can add multiple elements
✔ Can add text also
✔ Does NOT return element

Example:
box.append(newP);
box.append("Some text");


====================================================
4) prepend(element)
----------------------------------------------------
Adds element as FIRST child of parent.

Example:
box.prepend(newP);


====================================================
5) insertAdjacent(where, element)
----------------------------------------------------
Inserts element at specific position.

Syntax:
element.insertAdjacentElement(position, newElement);

Positions:

"beforebegin"   → Before element
"afterbegin"    → Inside element (start)
"beforeend"     → Inside element (end)
"afterend"      → After element


Example:
box.insertAdjacentElement("beforebegin", newP);
box.insertAdjacentElement("afterbegin", newP);
box.insertAdjacentElement("beforeend", newP);
box.insertAdjacentElement("afterend", newP);


====================================================
ADDING BUTTON EXAMPLE
====================================================

let btn = document.createElement('button');
console.dir(btn);
→ button

btn.innerText = "click me!";
→ 'click me!'

let box = document.queryselector('.box')
box.appendChild(btn);
→ <button>click me!</button>


====================================================
KEY DIFFERENCE
====================================================

appendChild()  → Only 1 element, returns element
append()       → Multiple items, can add text, no return
prepend()      → Adds at beginning
insertAdjacent → Adds at specific position
createElement  → Creates element only (not added)

====================================================
====================================================
insertAdjacentElement() – POSITION STRINGS
====================================================

Used to insert an element relative to a target element.

SYNTAX
----------------------------------------------------
targetElement.insertAdjacentElement(position, element);


POSITION STRINGS (case-insensitive)
====================================================

1) "beforebegin"
----------------------------------------------------
→ Before the target element itself
→ Outside the element

Structure:

<button></button>
<p>Paragraph</p>

Example:
p.insertAdjacentElement('beforebegin', btn);

Result:
<button>NEW BUTTON!!!</button>
<p>Paragraph</p>


----------------------------------------------------

2) "afterbegin"
----------------------------------------------------
→ Inside the target element
→ Before its FIRST child

Structure:

<p>
    <button>NEW BUTTON!!!</button>
    existing content
</p>

Example:
p.insertAdjacentElement('afterbegin', btn);


----------------------------------------------------

3) "beforeend"
----------------------------------------------------
→ Inside the target element
→ After its LAST child

Structure:

<p>
    existing content
    <button>NEW BUTTON!!!</button>
</p>

Example:
p.insertAdjacentElement('beforeend', btn);


----------------------------------------------------

4) "afterend"
----------------------------------------------------
→ After the target element itself
→ Outside the element

Structure:

<p>Paragraph</p>
<button>NEW BUTTON!!!</button>

Example:
p.insertAdjacentElement('afterend', btn);


====================================================
VERY IMPORTANT
====================================================

✔ If same element (btn) is inserted again,
  it MOVES — it does NOT duplicate.

✔ Only one position at a time.

✔ Element must exist in DOM to use this properly.


====================================================
VISUAL MEMORY TRICK
====================================================

beforebegin  → Outside, before element
afterbegin   → Inside, start
beforeend    → Inside, end
afterend     → Outside, after element

====================================================
REMOVING ELEMENTS IN DOM
====================================================

1) removeChild(element)
----------------------------------------------------
Removes a child from a parent.
Must call on parent.

SYNTAX:
parent.removeChild(child);

Example:
body.removeChild(btn);
→ <button>NEW BUTTON!!!</button>

✔ Returns removed element
✔ Element must be direct child


----------------------------------------------------

2) remove()
----------------------------------------------------
Removes element directly.
No need to access parent.

SYNTAX:
element.remove();

Example:
btn.remove();
→ undefined

✔ Easier than removeChild()
✔ No return value


----------------------------------------------------

IMPORTANT NOTE
----------------------------------------------------
appendChild()  → Adds element
removeChild()  → Removes child from parent
remove()       → Removes element directly

If you remove an element,
you can add it again later.
