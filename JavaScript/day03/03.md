
TEMPLATE LITERALS
============================================================================================================================

Description:
Used to add embedded expressions inside a string.

Syntax:
`string text ${expression} string text`

Example:
let a = 5;
let b = 10;

console.log(`Your pay ${a + b} rupees.`);
// Output: Your pay 15 rupees

Additional Example:
let pencilPrice = 10;
let eraserPrice = 5;

let output = `The total price is : ${pencilPrice + eraserPrice} Rupees.`;
console.log(output);

Important Note:
- Uses backticks (` `)
- Allows variables and expressions inside ${}

CREATING JS FILE
============================================================================================================================

Create a file with ".js" extension.
Example:
hello.js


LINKING JS FILE TO HTML
============================================================================================================================

Description:
Use <script> tag to link external JS file to HTML.

Syntax:
<script src="path_to_file.js"></script>

Example (index.html):

<!DOCTYPE html>
<html>
<head>
    <title>JS Demo</title>
    <script src="./hello.js"></script>
</head>
<body>
</body>
</html>

Important Notes:
✔ Use src attribute to specify file path
✔ Best practice: Place <script> at end of <body> for better loading

COMPARISON OPERATORS IN JAVASCRIPT
============================================================================================================================

Definition:
Comparison operators compare two values and return true or false.

COMMON OPERATORS
==    → Equal (value only)
===   → Strict equal (value + type)
!=    → Not equal
!==   → Strict not equal
>     → Greater than
<     → Less than
>=    → Greater than or equal
<=    → Less than or equal

VALUE vs STRICT:
let x = 10;
let y = "10";

x == y    → true
x === y   → false

Best Practice:
✔ Always use === to avoid type coercion

============================================================================================================================
COMPARISON FOR STRINGS (NON-NUMBERS)
============================================================================================================================

Strings are compared using Unicode values (lexicographic order).
Comparison is character by character.

CASE SENSITIVITY:
'a' > 'A'   → true   (lowercase has higher Unicode value)

ALPHABETICAL ORDER:
'a' < 'b'   → true
'b' < 'c'   → true
'B' < 'C'   → true

SPECIAL CHARACTERS:
'*' < '&'   → false

IMPORTANT RULE:
"apple" < "banana"  → true
If first characters same, next characters are compared.

INTERVIEW SUMMARY:
Numbers → Numeric comparison
Strings → Unicode comparison
===     → Safer than ==
Lowercase > Uppercase

============================================================================================================================
CONDITIONAL STATEMENTS IN JAVASCRIPT
============================================================================================================================

1) IF
if (condition) {
    // code
}

Example:
if (age >= 18) {
    console.log("You can vote");
}

--------------------------------------------------

2) IF...ELSE
if (condition) {
    // true block
} else {
    // false block
}

--------------------------------------------------

3) IF...ELSE IF...ELSE
if (condition1) {
    // block 1
} else if (condition2) {
    // block 2
} else {
    // default block
}

--------------------------------------------------

4) SWITCH
Used when comparing one value with multiple fixed cases.

Syntax:
switch (expression) {
    case value:
        // code
        break;
    default:
        // code
}

Traffic Signal Example:
let color = "red";

switch (color) {
    case "red":
        console.log("Stop");
        break;
    case "yellow":
        console.log("Slow Down");
        break;
    case "green":
        console.log("Go");
        break;
    default:
        console.log("Invalid Signal");
}

Notes:
✔ Good for fixed value comparison
✔ Do not use for > or < comparisons
✔ Can be slightly faster for many fixed values


--------------------------------------------------

5) TERNARY OPERATOR
Short form of if...else.

Syntax:
condition ? valueIfTrue : valueIfFalse;

Example:
let result = (age >= 18) ? "Adult" : "Minor";

===========================================================================================================================
LOGICAL OPERATORS IN JAVASCRIPT
============================================================================================================================

1) AND (&&)
Returns true only if BOTH conditions are true.

Example:
if (age >= 18 && hasID) {
    console.log("Allowed");
}

--------------------------------------------------

2) OR (||)
Returns true if at least one condition is true.

Example:
if (isAdmin || isEditor) {
    console.log("Access granted");
}

--------------------------------------------------

3) NOT (!)
Reverses the result.

Example:
if (!isLoggedIn) {
    console.log("Please login");
}

INTERVIEW SUMMARY:
&&  → All must be true
||  → Any one true is enough
!   → Opposite result
