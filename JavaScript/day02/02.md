============================================================================================================================
JAVASCRIPT IDENTIFIERS
============================================================================================================================

Definition:
Identifiers are names given to variables, functions, classes, or objects in JavaScript.

------------------------------------------------------------------------------------------------
RULES FOR NAMING IDENTIFIERS
------------------------------------------------------------------------------------------------
1) Allowed: letters (a-z, A-Z), digits (0-9), underscore (_), dollar sign ($)
2) Must begin with letter, _ or $
   - Cannot start with digit
3) Case-sensitive (myVar and MyVar are different)
4) Cannot use reserved keywords (var, let, const, function, if, else, return, class)
5) No spaces or special characters (except _ and $)
6) Should be meaningful and readable (best practice)

------------------------------------------------------------------------------------------------
EXAMPLES
------------------------------------------------------------------------------------------------
Valid:
var myVar;
let _count;
const $price = 100;
function getData() {}
let α = 5;

Invalid:
var 1stValue;      // starts with digit
let my-var;        // contains hyphen
const var = 10;    // reserved keyword

------------------------------------------------------------------------------------------------
WAYS OF WRITING IDENTIFIERS
------------------------------------------------------------------------------------------------
1) camelCase   → myVariable, getUserData, totalAmount
2) PascalCase  → MyClass, UserProfile, EmployeeDetails
3) snake_case  → my_variable, total_amount, user_name


============================================================================================================================
BOOLEAN IN JAVASCRIPT
============================================================================================================================

Definition:
Boolean represents a logical data type with only two values: true or false.

Usage:
- Used in conditions, loops, comparisons.

Examples:
var isActive = true;
let hasAccess = false;
const isLoggedIn = true;

Key Points:
1) Only two values: true / false
2) Case-sensitive (True / FALSE invalid)
3) Returned by comparisons:
   5 > 3        → true
   10 === "10"  → false
4) Used in if statements and logical expressions


============================================================================================================================
WHAT IS TYPESCRIPT?
============================================================================================================================

Definition:
TypeScript is a superset of JavaScript developed by Microsoft.
It adds static typing and compiles to plain JavaScript.

Key Features:
✔ Static Type Checking (compile-time error detection)
✔ Better IDE support (autocomplete, refactoring)
✔ Modern ECMAScript features
✔ OOP support (classes, interfaces, modules)
✔ Gradual adoption in existing JS projects

Example:

JavaScript:
let age = 23;
age = "twenty-three";      // No error

TypeScript:
let age: number = 23;
age = "twenty-three";      // Error


============================================================================================================================
STRING IN JAVASCRIPT
============================================================================================================================

Definition:
A string is a sequence of characters used to represent text.

------------------------------------------------------------------------------------------------
DECLARATION
------------------------------------------------------------------------------------------------
Double quotes:
let name = "Ajay";

Single quotes:
let city = 'Delhi';

Backticks (Template Literals):
const greeting = `Hello, ${name}!`;

Examples:
let name = "Tony Stark";
let role = 'ironman';
let char = 'a';
let num = '23';
let empty = " ";

------------------------------------------------------------------------------------------------
KEY FEATURES
------------------------------------------------------------------------------------------------
1) Strings are immutable
2) Index access: name[0]
3) Length: name.length
4) Common Methods:
toUpperCase()        → Converts entire string to uppercase.
toLowerCase()        → Converts entire string to lowercase.

trim()               → Removes whitespace from both ends of string.
slice(start,end)     → Extracts part of string (end not included).
substring(start,end) → Extracts part of string (no negative indexes).

replace(old,new)     → Replaces first matching substring.
includes(str)        → Checks if substring exists (returns true/false).

split(separator)     → Splits string into array based on separator.
concat(str)          → Joins strings together.

charAt(index)        → Returns character at given index.
indexOf(str)         → Returns first index of substring (-1 if not found).
lastIndexOf(str)     → Returns last index of substring.


Note:// slice() vs substring()
Both extract part of a string. Main difference → handling of negative values

let str = "HelloWorld";

1) Negative Index

console.log(str.slice(-5));      
//"World"  ✔ Works (counts from end)

console.log(str.substring(-5));  
// "HelloWorld" ❗ Negative treated as 0

============================================================================================================================
COMMON STRING METHODS – ONE SHORT EXAMPLE EACH
============================================================================================================================

let str = "  Hello World  ";

str.toUpperCase();           // "  HELLO WORLD  "
str.toLowerCase();           // "  hello world  "

str.trim();                  // "Hello World"
str.slice(2,7);              // "Hello"
str.substring(2,7);          // "Hello"

str.replace("World","JS");   // "  Hello JS  "
str.includes("Hello");       // true

str.split(" ");              // ["", "", "Hello", "World", "", ""]
"Hello".concat(" ","JS");    // "Hello JS"

str.charAt(2);               // "H"
str.indexOf("World");        // 8
str.lastIndexOf("l");        // 11

============================================================================================================================
STRING INDICES & IMMUTABILITY
============================================================================================================================

Definition:
Strings are zero-based indexed (first character at index 0).

Example: "TONY STARK"

T=0  O=1  N=2  Y=3  (space)=4  S=5  T=6  A=7  R=8  K=9

Key Concepts:
✔ Zero-based indexing
✔ Spaces count
✔ Access using bracket notation: str[index]

Immutability:
name[0] = 'X';     // ❌ Not allowed
name = "Ravi";     // ✔ Allowed (reassignment)


============================================================================================================================
STRING CONCATENATION
============================================================================================================================

Definition:
Combining two or more strings into one.

Methods:
1) + operator
   firstName + " " + lastName

2) += operator
   msg += " World"

3) concat()
   "Hello".concat(" ", "World")

4) Template literals (preferred)
   `Hello ${name}, welcome!`


============================================================================================================================
NULL AND UNDEFINED
============================================================================================================================

1) undefined
   - Variable declared but not initialized
   - Default return value of functions without return

   let x;
   console.log(x);      // undefined

2) null
   - Intentional absence of value

   let y = null;

Key Differences:
- typeof undefined → "undefined"
- typeof null → "object"
- undefined → not assigned
- null → explicitly assigned no value
- Both are falsy

If string contains double quotes:

Avoids syntax errors without escape characters.

{}
function(){}

if ("Hello") {
    console.log("Truthy");  // Executes
}

console.log(Boolean("0"));   // true
console.log(Boolean([]));    // true
