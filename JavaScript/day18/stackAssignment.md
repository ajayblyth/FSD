========================================================================================================================
STACK DATA STRUCTURE IMPLEMENTATION IN JAVASCRIPT
========================================================================================================================

DEFINITION
------------------------------------------------------------------------------------------------------------------------
Stack is a linear data structure that follows the LIFO principle:

                                Last In First Out

The last element inserted into the stack is the first one removed.


STACK OPERATIONS
------------------------------------------------------------------------------------------------------------------------
1) push(element)  → Add element to the top of stack
2) pop()          → Remove element from the top
3) peek()         → View top element without removing
4) isEmpty()      → Check if stack is empty
5) size()         → Number of elements in stack
6) print()        → Display stack elements



========================================================================================================================
IMPLEMENTATION USING ARRAY
========================================================================================================================

class Stack {

    constructor(){
        this.items = [];
    }

    // PUSH → add element
    push(element){
        this.items.push(element);
    }

    // POP → remove element
    pop(){
        if(this.isEmpty()){
            return "Stack Underflow";
        }
        return this.items.pop();
    }

    // PEEK → top element
    peek(){
        return this.items[this.items.length - 1];
    }

    // CHECK EMPTY
    isEmpty(){
        return this.items.length === 0;
    }

    // SIZE
    size(){
        return this.items.length;
    }

    // PRINT STACK
    print(){
        console.log(this.items.join(" "));
    }
}



========================================================================================================================
USAGE EXAMPLE
========================================================================================================================

let stack = new Stack();

stack.push(10);
stack.push(20);
stack.push(30);

stack.print();


OUTPUT
------------------------------------------------------------------------------------------------------------------------
10 20 30


TOP ELEMENT
------------------------------------------------------------------------------------------------------------------------

console.log(stack.peek());

Output
30



REMOVE ELEMENT
------------------------------------------------------------------------------------------------------------------------

stack.pop();

stack.print();

Output
10 20



STACK SIZE
------------------------------------------------------------------------------------------------------------------------

console.log(stack.size());

Output
2



CHECK EMPTY
------------------------------------------------------------------------------------------------------------------------

console.log(stack.isEmpty());

Output
false
