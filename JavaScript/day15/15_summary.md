Events are signals that something has occurred. (user inputs / actions)

onclick (when an element is clicked)

onmouseenter (when mouse enters an element)

What are DOM Events?
Think of DOM Events as how your web browser and JavaScript talk to each other to make a webpage interactive.

The DOM (Document Object Model) is like a map the browser creates for your webpage, showing all its parts like text, images, and buttons.

Events are actions that happen on this webpage map. These can be things you do, like clicking a button, typing in a search box, or moving your mouse around.

JavaScript is a programming language that can "listen" for these events. When an event happens, JavaScript can jump into action and run specific code.

This "listening and reacting" is exactly how websites become interactive! It's what makes things happen when you click, type, or interact with a page.
